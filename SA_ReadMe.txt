
To setup:

1. Run the SA-makedb.php to create the tables and insert the data. 
	Ensure you have df_diseases.txt in the same directory
	
2. Start on SA-search.php 

3. Enter a search term or phrase

4. Click submit and list a list of Diseases by Name and Symptoms is listed

5. Clicking on the name of the disease will take you to the next page that
prints out other fields about the disease
